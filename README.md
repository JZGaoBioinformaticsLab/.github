# Jianzhao Gao Bioinformatics Lab

We develop mathematical, statistical, and AI methods for bioinformatics, with applications in structural bioinformatics, single-cell genomics, proteomics, network modeling, and topological deep learning.

## Research Areas

- Structural bioinformatics
- Single-cell genomics
- Proteomics and disease risk prediction
- Network biology
- Topological data analysis and topological deep learning
- Biomedical AI
